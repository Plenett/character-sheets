# Fiche Personnage "Le Secret Des Gardiens" pour Roll20

Cette fiche personnage est basée sur le travail d'Ezechiel Din Cyr (@EzechielDinCyr).

## Création des dés

Vous devrez créer des dés personnalisés pour faire fonctionner les jets de dés automatique.

Vous allez devoir créer 7 dés avec les statistiques suivantes:

* Nom - Probabilité face "0" - Probabilité face "1"
* 0 - 7 - 3
* 1 - 6 - 4
* 2 - 5 - 5
* 3 - 4 - 6
* 4 - 3 - 7
* 5 - 2 - 8
* 6 - 0 - 10
